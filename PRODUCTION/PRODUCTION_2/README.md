## 🚀 Quick Start

### Prerequisites
- Go 1.21+
- Node.js 22+
- MongoDB 7+

### Setup

**Backend:**
```bash
cd PRODUCTION_2/backend
go mod download
cp .env.example .env
go run cmd/server/main.go
```

**Frontend:**
```bash
cd PRODUCTION_2/frontend
npm install
cp .env.example .env
npm run dev
```

### Access
- Frontend: http://localhost:3000
- Backend: http://localhost:8080
- API Docs: http://localhost:8080/swagger
