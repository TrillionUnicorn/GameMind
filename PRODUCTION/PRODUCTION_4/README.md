## 🚀 Quick Start

### Prerequisites
- Node.js 22+
- Python 3.11+
- PostgreSQL 16+

### Setup

**Backend:**
```bash
cd PRODUCTION_4/backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m uvicorn app.main:app --reload
```

**Frontend:**
```bash
cd PRODUCTION_4/frontend
npm install
ng serve
```
